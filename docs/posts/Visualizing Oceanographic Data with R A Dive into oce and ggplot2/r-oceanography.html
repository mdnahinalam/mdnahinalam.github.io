üåä Visualizing Oceanographic Data with R: A Dive into oce and ggplot2

Oceanographic data, such as measurements from Conductivity-Temperature-Depth (CTD) instruments, are vital for understanding marine environments. In this post, we‚Äôll explore how to use R‚Äôs oce package to handle such data and employ ggplot2 for insightful visualizations.‚Äã

üìä Visualizing with ggplot2

While oce provides its own plotting functions, integrating with ggplot2 offers enhanced customization. To use ggplot2, we‚Äôll convert the CTD object into a data frame:

Code
# Convert CTD object to a data frame
ctd_df <- as.data.frame(ctd[["data"]])

üå°Ô∏è Temperature vs. Depth

Let‚Äôs plot temperature against depth:

Code
ggplot(ctd_df, aes(x = temperature, y = pressure)) +
  geom_line(color = "blue") +
  scale_y_reverse() +  # Depth increases with pressure
  labs(title = "Temperature Profile",
       x = "Temperature (¬∞C)",
       y = "Pressure (dbar)") +
  theme_minimal()

üßÇ Salinity vs. Depth

Similarly, plot salinity against depth:

Code
ggplot(ctd_df, aes(x = salinity, y = pressure)) +
  geom_line(color = "darkgreen") +
  scale_y_reverse() +
  labs(title = "Salinity Profile",
       x = "Salinity (PSU)",
       y = "Pressure (dbar)") +
  theme_minimal()

üß™ Calculating Derived Parameters

Code
# Extract salinity, temperature, and pressure from the CTD object
salinity <- ctd[["salinity"]]
temperature <- ctd[["temperature"]]
pressure <- ctd[["pressure"]]

# Calculate potential temperature referenced to the surface (0 dbar)
theta <- swTheta(salinity, temperature, pressure, referencePressure = 0)

# Add the calculated potential temperature to the CTD object
ctd <- oceSetData(ctd, name = "theta", value = theta)

# Now you can access the potential temperature from the CTD object
